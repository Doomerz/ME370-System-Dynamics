# Rules
1. There can be no hope.. or loops
2. Every node must be connected

# Form from linear graph using these steps:
1. include all nodes
2. Include all across-variable sources
3. Include as many as possible A-type elements
4. Include as many as possible D-type elements
5. Include as many as possible T-type elements

# further
edges in the normal tree are branches, inter-node connections that aren't branches are called links.

A-type = Accumulators (across) storing potential energy
D-type = Dissipators
T-type = transducers/transformers {through} (like inductors the imply inertial flow storing kinetic energy)
